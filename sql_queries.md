## üè¢ Administraci√≥n

### Departamentos
```sql
SELECT ID_DEPARTAMENTO AS "ID_DEPARTAMENTO", NOMBRE_DEPARTAMENTO AS "NOMBRE_DEPARTAMENTO" FROM DEPARTAMENTO
```

### Municipios
```sql
SELECT ID_MUNICIPIO AS "ID_MUNICIPIO", ID_DEPARTAMENTO AS "ID_DEPARTAMENTO", NOMBRE_MUNICIPIO AS "NOMBRE_MUNICIPIO" FROM MUNICIPIO
```

### Sucursales
```sql
SELECT ID_SUCURSAL AS "ID_SUCURSAL", ID_MUNICIPIO AS "ID_MUNICIPIO", DIRECCION AS "DIRECCION", TIPO AS "TIPO" FROM SUCURSAL
```

### Puestos

### Empleados
